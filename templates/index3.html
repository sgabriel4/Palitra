Palitra

Palitra (from Russian: palette) is a project created by Deni Shakhbulatov that lets you upload the picture and displays the color pallete of it. The colors that appear the most will be displayed. The program uses Flask, Scipy, Pillow and Numpy, if you are curious how it works, check the Github repo. Supported formats: png, jpeg, jpg.
